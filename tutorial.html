üöÄ Quick Start Guide (4 Simple Steps)

1

üñ±Ô∏è Copy the Passenger List

On a Computer (PC/Mac):

  1. Open BPM and go to your flight
  2. Click anywhere in the passenger list
  3. Press Ctrl+A (Cmd+A on Mac) to select everything
  4. Press Ctrl+C (Cmd+C on Mac) to copy

On Your Phone:

  1. Android users: Install "XClipper" from Play Store first
  2. In BPM, tap and hold on the word "FLT NBR"
  3. Drag the blue handles to select everything from "FLT NBR" down to "Showing X entries"
  4. Tap "Copy" or use XClipper's copy button
2

üìã Paste Here

  1. Click the big white box that says "Paste passenger manifest here..."
  2. On Computer: Press Ctrl+V (Cmd+V on Mac)
  3. On Phone: Tap and hold in the box, then tap "Paste"
  4. You should see passenger names, seats, and meal codes appear
3

üîß Generate Report

  1. Click the red "Generate Report" button
  2. OR press Ctrl+Enter (on computer) for quick shortcut
  3. Wait 2 seconds - the report will appear below
  4. You'll see a ‚úÖ success message when done
4

üñ®Ô∏è Print

  1. Click the green "Print Report" button
  2. In the print window, check "All pages"
  3. Set "Scale" to "Fit to page" (important!)
  4. Click "Print"

üì± Platform-Specific Guides

ANDROID Must use XClipper

Due to Android copy limit restrictions:

  1. Long press on "FLT NBR"
  2. Select all text by dragging handles
  3. Use "XCopy" from XClipper app
  4. Or drag to select all passenger data

iOS Standard Selection

For iPhone/iPad users:

  1. Tap and hold on "FLT NBR"
  2. Drag selection handles to select from "FLT NBR" until "Showing X entries"
  3. Copy the selected text

WINDOWS/MAC Keyboard & Mouse

For desktop computers:

  1. Click before "FLT NBR"
  2. Press Ctrl+A (Cmd+A) to select all
  3. Press Ctrl+C (Cmd+C) to copy
  4. Paste in the textbox with Ctrl+V (Cmd+V)

‚ùì Common Problems & Solutions

"Nothing happens when I click Generate"
Fix: Make sure you copied the ENTIRE passenger list from "FLT NBR" down to "Showing X entries". The tool needs the complete data to work.
"On Android, it says text is too long"
Fix: You MUST install "XClipper" from Play Store. Android has a copy limit that XClipper fixes. This is the only way for Android to copy long manifests.
"The report is too small to read"
Fix: Click "Advanced Settings" ‚Üí Increase "Font Size" to 12px or 14px ‚Üí Click "Print Report with Current Settings".
"Business class meals don't show"
Fix: This is normal! Business class has blank lines so you can write in the standard meals (chicken/beef/fish). The tool only shows special meals automatically.
"Only shows 1-2 passengers but I know there are more special meals"
Fix: The tool only shows passengers with special meals or PLAT/EMER status. Regular passengers without special meals/status won't appear.
"Can't copy from BPM on iPhone"
Fix: On iPhone, tap and hold on "FLT NBR", then slowly drag the blue selection handles down to the bottom. Be patient and precise - it takes practice!
"Seat chart doesn't look right"
Fix: Click "Advanced Settings" ‚Üí Make sure the correct aircraft type is selected. For 737 choose "737NG", for A330 choose "A333", etc.
"Print preview shows only 1 page but I have 200 passengers"
Fix: In print preview, check if you're printing "All pages" not "Current page". The report creates multiple pages automatically.
"The report doesn't show infant meals (BBML)"
Fix: BBML DOES show! It appears as "+BBML" below the parent's meal code in the list view. In seat chart, it shows as "MLCODE:BBML".
"On PC, Ctrl+Enter doesn't work"
Fix: Make sure you click inside the text box first, then press Ctrl+Enter. If using Mac, use Cmd+Enter.
"Report shows empty seat numbers (01A, 02B, etc.)"
Fix: This is a FEATURE, not a bug! Empty seats show seat numbers so you can easily see which seats are available.
"Colors don't print on my printer"
Fix: In print settings, make sure "Print background colors and images" is checked. Or use the "Aft Perspective View" which uses black text only.
"Need to print for multiple flights"
Fix: Clear the text box after each print (click "Clear" button), then paste the next flight's manifest and repeat.
"Seat chart shows all seats as 'occupied'"
Fix: Uncheck "Show All Business Class Passengers" in Advanced Settings. This setting shows ALL business passengers (even without special meals).
"Can't download the offline version"
Fix: Some phones block downloads. Try on a computer, or save the webpage directly from your browser menu (3 dots ‚Üí Save page).
"Report missing middle seats in business class"
Fix: 737 business class has no B and E seats! This is correct - seats are A, C, D, F only in 737 business class.
"Want to see the view from the back of the plane"
Fix: Check "Aft Perspective View" in Advanced Settings, then print. Turn the paper 180¬∞ for perfect back-to-front view!
"Report is cutting off on the right side when printing"
Fix: In print settings, change "Scale" to "Fit to page" or reduce "Table Width" in Advanced Settings to 90%.
"Font is too big and the report is too many pages"
Fix: Use "Compact Print Mode" in Advanced Settings ‚Üí Print Layout Options to reduce pages by 20-30%.
"The tool says 'No passengers found' but I pasted the manifest"
Fix: You might have copied the wrong section. Make sure you copy from the actual passenger list, not the flight info header.
"I need to show regular business class passengers too"
Fix: Check "Show All Business Class Passengers" in Advanced Settings. This will show ALL business class passengers, not just special meals.
"The page reloads and I lose my data"
Fix: Use "Download Offline Version" button to save a local copy. Then you can use it without internet and your data won't be lost.
"I accidentally changed settings and now it looks wrong"
Fix: Click "Reset to Defaults" button in Advanced Settings to return to automatic settings.
